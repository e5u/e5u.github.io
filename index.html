使用 Koyeb 部署 Melody 音乐精灵

方式一：一键部署

方式二：手动部署

一、注册 Koyeb 账号

二、点击 “+ Create App”

三、输入你想要的三级域名前缀

四、配置

选择 Docker

Docker image 输入 foamzou/melody:latest

Environment variables 可以删除，本次部署不需要

Ports 如图设置

Scale 数目设置为 1

Instance size 可以选择 MICRO

Region 选择 par (法国)

Service name 取一个简短的名字，方便进入

五、Create Service !

等待状态从 Pending 到 Healthy 后 点击 Pubic URL 进入 Melody

默认 Melody Key 是 melody

部署完成，此文完。
